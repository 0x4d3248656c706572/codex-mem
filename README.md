# codex-mem

Persistent memory for Codex.

## Install

```bash
npx codex-mem install --ide codex-cli
```

## Start

```bash
npx codex-mem start
```

## Use

Run Codex normally.

The package installs:
- Codex hooks
- Codex MCP config
- transcript watching
- AGENTS context injection

## Paths

- Codex config: `~/.codex`
- codex-mem data: `~/.codex-mem`
- viewer: `http://127.0.0.1:37777`
